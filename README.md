# 📱 QR Make App - Генератор QR-кодов

![QR Make App Logo](qr_icon.png)

**QR Make App** — это удобное десктопное приложение для мгновенного создания QR-кодов из любой текстовой информации. Просто введите ссылку или текст и получите готовый QR-код в один клик!

## ✨ Особенности

### 🎯 Основные возможности
- **Мгновенная генерация** - создавайте QR-коды за секунду
- **Работает офлайн** - не требует подключения к интернету
- **Автоматическая организация** - все коды сохраняются в отдельную папку
- **Предпросмотр** - сразу видите результат перед сохранением
- **Быстрый доступ** - удобная навигация по созданным кодам

### 🎨 Интерфейс
- **Темная тема** - снижает нагрузку на глаза при длительной работе
- **Минималистичный дизайн** - ничего лишнего
- **Интуитивно понятно** - разберется любой пользователь

### 📦 Что можно кодировать
- Ссылки на сайты
- Текстовую информацию
- Контактные данные
- Wi-Fi пароли
- И многое другое!
